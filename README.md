# 🐾 PawGo - Doorstep Pet Grooming

A modern, responsive website for **PawGo**, a professional pet grooming service that brings grooming to your doorstep.

## Features

✨ **Modern Design** - Clean, professional interface with smooth animations
📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
🎯 **User-Friendly Navigation** - Easy-to-use navigation menu with smooth scrolling
📋 **Online Booking** - Simple appointment booking form
💬 **WhatsApp Integration** - Direct messaging button for customer inquiries
⭐ **Customer Reviews** - Testimonials section showcasing happy customers
💳 **Service Pricing** - Clear pricing for all grooming packages

## Services Offered

- **🛁 Basic Bath** - ₹499 (Bath, drying and brushing)
- **✂️ Full Grooming** - ₹999 (Bath, haircut, nail trim and ear cleaning)
- **💎 Premium Spa** - ₹1499 (Luxury spa treatment and styling)
- **🐱 Cat Grooming** - ₹899 (Special care grooming for cats)

## How to Use

1. Open `index.html` in your web browser
2. Browse through sections:
   - **Services** - View all available grooming packages
   - **How It Works** - Learn the simple booking process
   - **Reviews** - Read customer testimonials
   - **Book** - Schedule your appointment
